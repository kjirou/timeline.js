timeline.js の紹介

はじめに

  • 複数の非同期処理を連携するためのJavaScriptライブラリです
  • 非同期処理同士の分離性を高めることと、連携状況をわかり易くすることを主な目的としています
  • Flashのタイムライン編集機能をヒントにした実装をしています
  • JSDeferredに激しく依存しています
  • 動作はWindows7のIE8/Firefox3/Chromeのみで確認していますが、多分Macでも動くと思います
  • GitHub

ダウンロード・インストール

  • 1. JSDeferredのダウンロード をします
  • 2. timeline.jsのダウンロード をします
  • 3. jsdeferred.js を先に timeline.js を後に読み込みます、このページのHTMLソースを参考にして下さい

マニュアル

  • マニュアルは無いので、まずは 当ページで読み込んでいるサンプルコード をご覧下さい

基本的な使い方

  • クリックするとコンソールに結果が出力されます
  • 最も基本的な使い方
    var timeline = Timeline.factory(再生時間);
    var layer = timeline.createLayer();
    layer.onstart(コールバック関数);
    layer.on(開始する時間:ミリ秒);
    timeline.onfinish(コールバック関数);
    timeline.start();
  • 開始条件にレイヤ終了を指定
    layer.after(自分の前に実行するlayer);
    layer.wait(実行前の待機時間:ミリ秒);
  • chainを使った一括設定
    layer1.chain(layer2, layer3, ..);
  • 複数レイヤを終了条件へ指定
  • ありがちな設定間違いとエラー確認

JSDeferredを知るとより便利

  • レイヤ内で終了を遅延させる
  • 複数のタイムラインを同期
